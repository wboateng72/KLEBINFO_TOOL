This docker was created by William Boateng for the purpose of Klebsiella analysis (2025)

The pipeline contains the tools:

1. Kleborate 
2. Abricate
3. Bactinspector
4. Parsnp

This tool enables you to characterize your Klebsiella genomes by performing speciation, sequence typing,
serotyping, plasmid analysis, virulence analysis, resistance analysis and phylogenetic analysis
